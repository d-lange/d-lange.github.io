Mentorship

Web Application for professionals offering and seeking mentorship!

Navigating careers can be tricky. Find your way with Mentorship!

An application built to connect professionals that are seeking and offering mentorship to further improve in their goals, cultivate and guide talent, and gain insights into new industries.

Built with Ruby on Rails, PostgreSQL, LinkedIn API (OmniAuth), jQuery, Foundation, HTML, and CSS.

SnapShort

Social Web Application for short stories

A short story written in a snap! View a prompt and let yourself write what comes to mind. There is no time limit, but the challenge is to create a micro short story from limited rules.

Users can also leave actionable, specific and kind feedback through comments and vote on stories they like the most!

Built with Ruby, Sinatra, PostgreSQL, jQuery, HTML, CSS, RESTful design, and ActiveRecord polymorphic associations.
